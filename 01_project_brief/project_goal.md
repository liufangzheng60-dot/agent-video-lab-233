# 项目目标

`agent-video-lab` 的目标是建立一个面向 TikTok 商品视频的 AI 自动化剪辑学习与实验工作区。

核心闭环：

```text
AI 生成视频素材
-> Agent 盘点素材
-> 输出剪辑策略
-> 生成 CapCut 时间轴
-> 自动合成 final.mp4
-> 人工审片
-> TikTok 发布与数据反馈
```

本项目的重点不是泛泛学习工具，而是把 Agent、Skills、Vibe Coding 用到一个可以反复测试、复盘、改进的商品视频 MVP 中。

第一阶段成功标准：

- 能清楚管理商品素材、产品 brief、参考视频和输出结果。
- 能让 Agent 基于素材生成结构化盘点。
- 能产出可人工复刻的剪辑策略和时间轴。
- 能保留每次实验结果，方便复盘。
