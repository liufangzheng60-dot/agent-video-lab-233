# MVP 范围

## 包含

- 单个 TikTok 商品视频案例的素材整理。
- 素材盘点结果，包含图片、视频、AI 生成片段、参考视频。
- 商品卖点、痛点、使用场景、目标受众整理。
- TikTok 商品视频剪辑策略。
- 自定义 JSON 时间轴。
- CapCut 人工复刻用 CSV。
- 最终视频人工审片记录。
- 发布表现手动记录与复盘。

## 第一阶段输出

- `outputs/material_inventory/`
- `outputs/edit_strategy/`
- `outputs/timelines/timeline.json`
- `outputs/timelines/capcut_timeline.csv`
- `outputs/reports/`

## 暂不包含

- 自动发布 TikTok。
- 复杂素材理解模型。
- Web UI。
- 数据库。
- 多账号运营系统。
- 完整 SaaS 化产品。
