# P1-001 Publish Record

## Goal

Create manual TikTok publish tracking and performance feedback templates for `video_v001`.

## Inputs

- `03_tk_video_agent/outputs/renders/final_subtitled.mp4`
- `03_tk_video_agent/outputs/subtitles/subtitles.srt`
- `03_tk_video_agent/outputs/reports/manual_review.md`
- `03_tk_video_agent/outputs/timelines/capcut_timeline.csv`

## Outputs

- `05_final_outputs/publish_records/video_v001_publish_record.md`
- `05_final_outputs/performance_feedback/video_v001_feedback.md`

## Scope

- Manual publishing record only.
- No TikTok auto-publish.
- No external API calls.
- No video modification.
- No re-rendering.
- No dependency installation.

## Iteration Rules

- If 3-second retention is poor, improve the Hook first.
- If clicks are poor, improve subtitles and CTA first.
- If completion is poor, shorten pacing first.
- If comments ask about product function, strengthen demo and proof.
- If views are low but engagement is good, keep the structure and replace material.
