# Desktop To Laptop Handoff Checklist

- [ ] GitHub push succeeded.
- [ ] `.gitignore` protects `outputs` and `raw_videos`.
- [ ] requirements file is present.
- [ ] `.env.example` exists.
- [ ] Laptop has Python installed.
- [ ] Laptop has Git installed.
- [ ] Laptop has ffmpeg installed.
- [ ] Repo is cloned on laptop.
- [ ] `.venv` is created.
- [ ] smoke test passed.
- [ ] raw videos are manually copied or external disk is mounted.
- [ ] `git status --short` does not show generated media.
- [ ] User confirms no video artifacts will be committed.
