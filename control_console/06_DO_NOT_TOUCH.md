# Do Not Touch

The following files and directories must not be modified without explicit user approval:

- `control_console/`
- `00_references/`
- `products/*/assets/`
- `products/*/product_brief.md`
- `experiments/*/*/*/` records after manual data has been entered
- Any generated final video that has already been manually published or reviewed
- Any publish URL, performance data, or platform feedback entered by the user

## Special Rule

Do not cross-write between different `product_slug`, `sku_slug`, or `batch_id` folders.
